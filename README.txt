This code was written in python on VCS using the python interpreter. Version of 3.10.4 64bit.

How to run the code:
Once the python file is run; the user is prompted to enter their desired alphabet. Only letters in the latin
alphabet are allowed with any other characters otherwise being disregarded or leading to an error in the 
execution of the program. 

Tested cases:
abc ->	aaccbbcaba
ABc ->	AABBcBAccA
ABC ->	AABBCBACCA
440 ->	Incorrect input please try again
440" ->  Incorrect input please try again:
AB400c ->  AAcABccBBA (numbers disregarded as wrong)
ab ->  aabba
Ab ->  AAbbA
